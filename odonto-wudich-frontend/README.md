# Odonto Wudich - Frontend

Aplicação frontend em **React** para consumir a API do projeto Odonto Wudich (Django REST Framework).

## Tecnologias

- React
- Vite
- JavaScript (ES6+)

## Como rodar

```bash
cd odonto-wudich-frontend
npm.cmd install
npm.cmd run dev
```

Acesse em: `http://localhost:5173/`
